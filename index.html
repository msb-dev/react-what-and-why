A brief history of the web

Invented by Tim Berners-Lee in 1989 at CERN

Designed for publishing and linking articles

Berners-Lee created:

  • a web server
  • a web browser
  • HTTP to communicate
  • HTML

Static pages

HTML files were stored on web servers

A server could be browsed like a file system

Content was static and read-only

Life was simple! üòÄ

Server-side rendering

Later, user generated data was made possible

Data could be entered by the user via "forms"

The server would run code to generate dynamic pages based on the user's data

Problem: full page refreshes

Javascript

Websites became interactive. JS can edit the DOM (Document Object Model) directly within browsers

XMLHttpRequest allowed JavaScript to get and send data to the server

Now user generated data was possible without full page refreshes

Browsers with JavaScript became widely used and more powerful

Props

Props allow us to pass data into a component

State

State is a way to store data in React

State gets passed down into child components; it can't be passed up the tree

Lifting up state

State is always "owned" by a single component

If you need state higher up, move the useState to a parent or grandparent component

Only lift it as high as it needs to go
